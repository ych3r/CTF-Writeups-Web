# CTF Write-ups

## Idea behind this

Create a platform where people can share CTF write-ups.

## User Roles

- Moderator: can delete inappropriate write-ups.
- User: can publish and like write-ups.
- Admin: {...User, ...Moderator, can delete user or give user more privilege.}
